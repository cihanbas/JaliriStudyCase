# Welcome to Jaliri Study Case : )

**You can follow the steps below to run the application**

Step 1: install dependency packages

    yarn install

Step 2: install pods

    cd ios && pod install && cd ..

Step 3: start terminal

    yarn start

Step 4: open another terminal and **for ios** run ios device

    yarn ios

**for android** run android device

    yarn android

Happy Coding
